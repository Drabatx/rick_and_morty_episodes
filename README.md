# Rick And Morty Characters

Rick and Morty es una aplicacion que consume el api de Rick and Morty

## Installation

Usa git para descargar el proyecto

```bash
git clone https://github.com/Drabatx/rick_and_morty_episodes.git
```

## Description

Este proyecto usa Android Jetpack Compose para las vistas de personajes y episodios, se uso Flows para el consumo de servicios y la implementacion de patron de dise√±o Builder e Inyeccion de dependencias.
